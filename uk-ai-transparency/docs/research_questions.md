# Research Questions

This file lists research questions and motivations for the UK AI transparency prototype.

- What decision factors does automated decisioning use for high-stakes outcomes?
- Which model features correlate most with adverse decisions (decline/review)?
- Can we provide concise human-readable explanations for decisions derived from both structured fields and free text notes?
- How should a tamper-evident ledger be designed for auditability of decisions and explanations?
